

# 📘 UNIT – 10 : FILE HANDLING IN PHP (DETAILED)

File handling allows PHP to **read, write, upload, and manage files** on the server.

👉 Used in:

* File uploads (images, PDFs)
* Logs
* Data storage
* Reports
* Resume upload systems

---

## 1️⃣ What is File Handling?

### Definition (Exam-ready ✍️)

> File handling in PHP is the process of creating, reading, writing, and deleting files on the server.

---

## 2️⃣ Why File Handling is Needed?

* Store user data in files
* Upload profile images
* Save logs
* Read configuration files

---

## 3️⃣ File Modes in PHP (VERY IMPORTANT 🔥)

When opening a file, PHP needs a **mode**.

| Mode | Meaning                  |
| ---- | ------------------------ |
| `r`  | Read only                |
| `w`  | Write (overwrite)        |
| `a`  | Append                   |
| `r+` | Read + write             |
| `w+` | Read + write (overwrite) |
| `a+` | Read + append            |

---

## 4️⃣ Opening a File – `fopen()`

```php
<?php
$file = fopen("data.txt", "r");
?>
```

📌 If file does not exist:

* `r` → error
* `w` / `a` → creates file

---

## 5️⃣ Reading a File

### 🔹 `fread()`

```php
<?php
$file = fopen("data.txt", "r");
$content = fread($file, filesize("data.txt"));
echo $content;
fclose($file);
?>
```

---

### 🔹 `fgets()` (Line by line)

```php
<?php
$file = fopen("data.txt", "r");

while (!feof($file)) {
    echo fgets($file) . "<br>";
}

fclose($file);
?>
```

---

### 🔹 `file_get_contents()` (Simple way)

```php
<?php
echo file_get_contents("data.txt");
?>
```

---

## 6️⃣ Writing to a File

### 🔹 `fwrite()`

```php
<?php
$file = fopen("data.txt", "w");
fwrite($file, "Welcome to PHP File Handling");
fclose($file);
?>
```

⚠️ `w` mode **overwrites existing data**

---

### 🔹 Append Data

```php
<?php
$file = fopen("data.txt", "a");
fwrite($file, "\nNew line added");
fclose($file);
?>
```

---

## 7️⃣ Closing a File – `fclose()`

```php
fclose($file);
```

📌 Always close files to free memory.

---

## 8️⃣ Checking File Existence

```php
<?php
if (file_exists("data.txt")) {
    echo "File exists";
} else {
    echo "File not found";
}
?>
```

## 🔟 File Upload in PHP (VERY IMPORTANT 🔥)

### Step 1: HTML Form

```html
<form method="post" enctype="multipart/form-data">
    Select file:
    <input type="file" name="myfile">
    <input type="submit" name="upload">
</form>
```

⚠️ `enctype` is mandatory

---

### Step 2: PHP Upload Code

```php
<?php
if (isset($_POST["upload"])) {

    $fileName = $_FILES["myfile"]["name"];
    $tmpName  = $_FILES["myfile"]["tmp_name"];

    move_uploaded_file($tmpName, "uploads/" . $fileName);

    echo "File uploaded successfully";
}
?>
```

---

## 1️⃣1️⃣ `$_FILES` Array Structure

```php
$_FILES["myfile"]["name"];
$_FILES["myfile"]["type"];
$_FILES["myfile"]["size"];
$_FILES["myfile"]["tmp_name"];
$_FILES["myfile"]["error"];
```

---

## 1️⃣2️⃣ File Upload Validation (Security 🔐)

```php
<?php
$allowed = ["jpg", "png", "pdf"];
$ext = pathinfo($_FILES["myfile"]["name"], PATHINFO_EXTENSION);

if (in_array($ext, $allowed)) {
    move_uploaded_file($_FILES["myfile"]["tmp_name"], "uploads/" . $_FILES["myfile"]["name"]);
    echo "Valid file uploaded";
} else {
    echo "Invalid file type";
}
?>
```
