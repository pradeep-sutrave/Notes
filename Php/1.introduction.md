# 📘 UNIT – 1 : INTRODUCTION TO PHP (DETAILED)

---

## 1️⃣ What is PHP?

**PHP** stands for **Hypertext Preprocessor**.

👉 PHP is a **server-side scripting language**
👉 Used to create **dynamic web pages**
👉 Runs on the **server**, not in the browser

### Simple Definition (Exam-ready ✍️)

> PHP is a server-side scripting language used to develop dynamic and interactive web applications.

---

## 2️⃣ Why PHP is Needed?

HTML alone can create **static pages only**.

### Example Problem:

* Login system
* Showing user-specific data
* Storing data in database

❌ HTML cannot do this
✅ PHP can

So PHP is used to:

* Process forms
* Connect to databases
* Control application logic

---

## 3️⃣ Static Website vs Dynamic Website

| Static Website             | Dynamic Website        |
| -------------------------- | ---------------------- |
| Same content for all users | Content changes        |
| HTML, CSS                  | PHP, Database          |
| No login                   | Login, signup          |
| Fast but limited           | Powerful & interactive |

---

## 4️⃣ History of PHP

* **1994** – Created by **Rasmus Lerdorf**
* Initially called **Personal Home Page**
* Later renamed to **PHP: Hypertext Preprocessor**
* Open-source and free

👉 PHP evolved from small scripts to a **full backend language**.

## 7️⃣ Client–Server Architecture

### What is Client?

* Browser (Chrome, Firefox)

### What is Server?

* Computer that processes requests

### Flow:

```
Client (Browser)
     ↓
Server (PHP)
     ↓
Database
     ↓
Server
     ↓
Client
```

## 8️⃣ How PHP Works? (Core Concept 🔥)

### Step-by-Step:

1. User requests a `.php` page
2. Server receives request
3. PHP code executes on server
4. HTML output is generated
5. Browser displays output

⚠️ Browser **never sees PHP code**

## 9️⃣ PHP vs HTML vs JavaScript

| Feature         | PHP    | HTML    | JavaScript |
| --------------- | ------ | ------- | ---------- |
| Runs on         | Server | Browser | Browser    |
| Logic           | Yes    | No      | Yes        |
| Database access | Yes    | No      | No         |

## 🔟 PHP Installation (XAMPP)

To run PHP, you need:

* Web Server
* PHP Engine
* Database

### XAMPP provides:

* Apache (Server)
* PHP
* MySQL

### Folder Structure:

```
xampp/
 └── htdocs/
      └── your_project.php
```

Access via:

```
http://localhost/your_project.php
```
