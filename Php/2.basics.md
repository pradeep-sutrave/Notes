
# 📘 UNIT – 2 : PHP BASICS (DETAILED)

## 1️⃣ PHP Syntax

### Basic PHP Structure

```php
<?php
    // PHP code here
?>
```

✔ PHP code must be inside `<?php ?>`
✔ File extension must be `.php`

---

## 2️⃣ First PHP Program

```php
<?php
echo "Hello, PHP!";
?>
```

### Output:

```
Hello, PHP!
```

👉 `echo` is used to print output
👉 `;` ends a statement (mandatory)

---

## 3️⃣ PHP Comments

Used to explain code (important for exams & teamwork).

### Single-line

```php
// This is a comment
# This is also a comment
```

### Multi-line

```php
/*
 This is
 a multi-line
 comment
*/
```

---

## 4️⃣ Variables in PHP

A **variable** stores data.

### Rules:

* Starts with `$`
* Case-sensitive
* No data type declaration

```php
<?php
$name = "Bhupathi";
$age = 20;
$cgpa = 8.5;
?>
```

✔ Valid: `$name`, `$age`
❌ Invalid: `$1name`, `$my-name`

---

## 5️⃣ Data Types in PHP

PHP is **loosely typed**.

| Data Type | Example        |
| --------- | -------------- |
| String    | `"Hello"`      |
| Integer   | `25`           |
| Float     | `3.14`         |
| Boolean   | `true / false` |
| Array     | `[1,2,3]`      |
| NULL      | `null`         |

### Example

```php
<?php
$isLogin = true;
$price = 99.99;
?>
```

---

## 6️⃣ echo vs print

```php
<?php
echo "Hello";
print "World";
?>
```

| echo            | print           |
| --------------- | --------------- |
| Faster          | Slightly slower |
| Multiple values | Single value    |
| No return       | Returns 1       |

👉 **Use `echo` mostly**

---

## 7️⃣ Constants

Constants store **fixed values**.

```php
<?php
define("SITE_NAME", "My Website");
echo SITE_NAME;
?>
```

✔ Cannot be changed
✔ No `$` sign

---

## 8️⃣ Type Casting

Convert one data type to another.

```php
<?php
$a = "10";
$b = (int)$a;

echo $b; // 10
?>
```

---

## 9️⃣ PHP inside HTML (Very Important 🔥)

```php

<h2>
```
