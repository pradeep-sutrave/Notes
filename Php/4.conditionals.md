# рҹ”¶ PART A: CONDITIONAL STATEMENTS

Conditional statements are used to **make decisions**.

---

## 1пёҸвғЈ `if` Statement

Used when **only one condition** needs to be checked.

```php
<?php
$age = 20;

if ($age >= 18) {
    echo "Eligible to vote";
}
?>
```

рҹ“Ң If condition is false вҶ’ nothing happens.

---

## 2пёҸвғЈ `if вҖ“ else` Statement

Used when there are **two possible outcomes**.

```php
<?php
$marks = 35;

if ($marks >= 40) {
    echo "Pass";
} else {
    echo "Fail";
}
?>
```

---

## 3пёҸвғЈ `if вҖ“ elseif вҖ“ else` Statement

Used for **multiple conditions**.

```php
<?php
$score = 82;

if ($score >= 90) {
    echo "Grade A";
} elseif ($score >= 75) {
    echo "Grade B";
} elseif ($score >= 60) {
    echo "Grade C";
} else {
    echo "Fail";
}
?>
```

рҹ“Ң Conditions are checked **top to bottom**.

---

## 4пёҸвғЈ Nested `if`

An `if` inside another `if`.

```php
<?php
$age = 20;
$hasId = true;

if ($age >= 18) {
    if ($hasId) {
        echo "Entry allowed";
    } else {
        echo "ID required";
    }
} else {
    echo "Underage";
}
?>
```

---

## 5пёҸвғЈ `switch` Statement

Used when **one variable** is compared with **multiple fixed values**.

```php
<?php
$day = 3;

switch ($day) {
    case 1:
        echo "Monday";
        break;
    case 2:
        echo "Tuesday";
        break;
    case 3:
        echo "Wednesday";
        break;
    default:
        echo "Invalid day";
}
?>
```

рҹ“Ң `break` prevents fall-through.

---

## 6пёҸвғЈ Ternary Operator (Decision Shortcut)

```php
<?php
$age = 17;
echo ($age >= 18) ? "Adult" : "Minor";
?>
```

## 7пёҸвғЈ `for` Loop

Used when **number of iterations is known**.

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    echo $i . "<br>";
}
?>
```

---

## 8пёҸвғЈ `while` Loop

Used when **condition-based looping** is required.

```php
<?php
$i = 1;

while ($i <= 5) {
    echo $i . "<br>";
    $i++;
}
?>
```

вҡ пёҸ Missing increment вҶ’ infinite loop.

---

## 9пёҸвғЈ `doвҖ“while` Loop

Executes **at least once**, even if condition is false.

```php
<?php
$i = 6;

do {
    echo $i . "<br>";
    $i++;
} while ($i <= 5);
?>
```

---

## рҹ”ҹ `foreach` Loop

Used mainly for **arrays**.

```php
<?php
$colors = ["Red", "Green", "Blue"];

foreach ($colors as $color) {
    echo $color . "<br>";
}
?>
```

---

## 1пёҸвғЈ1пёҸвғЈ Loop Control Statements

### `break`

Stops loop execution.

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    if ($i == 3) {
        break;
    }
    echo $i;
}
?>
```

---

### `continue`

Skips current iteration.

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    if ($i == 3) {
        continue;
    }
    echo $i;
}
?>
```

---

## вңҚпёҸ UNITвҖ“4 EXAM SHORT NOTES

* Control statements control program flow
* `if` is used for decisions
* `switch` is used for fixed options
* Loops repeat code
* `foreach` is best for arrays
* `break` exits loop
* `continue` skips iteration
