
# 📘 UNIT – 6 : FUNCTIONS IN PHP (DETAILED)

Functions help us **reuse code**, **reduce repetition**, and **make programs structured**.

---

## 1️⃣ What is a Function?

### Definition (Exam-ready ✍️)

> A function is a block of code that performs a specific task and can be reused multiple times.

### Why Functions?

Without function:

```php
$a = 10 + 20;
$b = 10 + 20;
$c = 10 + 20;
```

With function:

```php
function add() {
    return 10 + 20;
}
```

✔ Less code
✔ Easy maintenance
✔ Reusability

---

## 2️⃣ Types of Functions in PHP

PHP supports **two main types**:

1️⃣ Built-in Functions
2️⃣ User-defined Functions

---

## 3️⃣ Built-in Functions

These are **already provided by PHP**.

### Examples

```php
echo strlen("PHP");      // 3
echo count([1,2,3]);    // 3
echo date("d-m-Y");
```

Some common built-in functions:

* `strlen()`
* `count()`
* `date()`
* `sqrt()`
* `max()`, `min()`

---

## 4️⃣ User-Defined Functions

Functions created by the programmer.

### Syntax

```php
function functionName() {
    // code
}
```

### Example

```php
<?php
function greet() {
    echo "Welcome to PHP";
}

greet(); // function call
?>
```

---

## 5️⃣ Function with Parameters

Parameters allow passing values to functions.

```php
<?php
function greet($name) {
    echo "Hello $name";
}

greet("Bhupathi");
?>
```

---

## 6️⃣ Function with Return Value

Functions can **return a value**.

```php
<?php
function add($a, $b) {
    return $a + $b;
}

$sum = add(10, 20);
echo $sum;
?>
```

📌 `return` sends value back to caller.

---

## 7️⃣ Default Argument Functions

If argument is not passed, **default value is used**.

```php
<?php
function greet($name = "Guest") {
    echo "Hello $name";
}

greet();
greet("Bhupathi");
?>
```

---

## 8️⃣ Variable Length Arguments

Allows passing **multiple values**.

```php
<?php
function total(...$numbers) {
    return array_sum($numbers);
}

echo total(10, 20, 30);
?>
```

---

## 9️⃣ Call by Value & Call by Reference

### Call by Value (Default)

```php
<?php
function change($x) {
    $x = 100;
}

$a = 10;
change($a);
echo $a; // 10
?>
```

---

### Call by Reference

```php
<?php
function change(&$x) {
    $x = 100;
}

$a = 10;
change($a);
echo $a; // 100
?>
```

📌 `&` makes changes permanent.

---

## 🔟 Recursive Functions

A function that **calls itself**.

### Example: Factorial

```php
<?php
function factorial($n) {
    if ($n == 1)
        return 1;
    else
        return $n * factorial($n - 1);
}

echo factorial(5);
?>
```

---

## 1️⃣1️⃣ Anonymous Functions (Intro)

Functions **without name**.

```php
<?php
$greet = function($name) {
    return "Hello $name";
};

echo $greet("PHP");
?>
```
