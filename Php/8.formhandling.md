## 2️⃣ HTML Form Basics

### Basic Form Structure

```html
<form method="post" action="process.php">
    Name: <input type="text" name="name"><br>
    Email: <input type="email" name="email"><br>
    <input type="submit" value="Submit">
</form>
```

📌 `name` attribute is **mandatory**
📌 `action` = PHP file
📌 `method` = GET or POST

---

## 3️⃣ Handling Form Using `$_POST`

### `process.php`

```php
<?php
$name  = $_POST["name"];
$email = $_POST["email"];

echo "Name: $name<br>";
echo "Email: $email";
?>
```

✔ Data hidden
✔ Secure for sensitive info

---

## 4️⃣ Handling Form Using `$_GET`

```html
<form method="get" action="process.php">
```

```php
<?php
echo $_GET["name"];
?>
```

⚠️ Data visible in URL
❌ Not for passwords

---

## 5️⃣ Checking Request Method (`$_SERVER`)

Very important for **security**.

```php
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    echo "Form submitted using POST";
}
?>
```

✔ Prevents direct URL access
✔ Best practice

---

## 6️⃣ Required Field Validation

```php
<?php
if (empty($_POST["name"])) {
    echo "Name is required";
} else {
    echo "Welcome " . $_POST["name"];
}
?>
```

📌 `empty()` checks blank input
📌 `isset()` checks variable existence

---

## 7️⃣ Input Sanitization (🔥 VERY IMPORTANT)

Protects from **malicious input**.

```php
<?php
$name = htmlspecialchars($_POST["name"]);
$email = filter_input(INPUT_POST, "email", FILTER_SANITIZE_EMAIL);

echo $name . "<br>";
echo $email;
?>
```

✔ Prevents XSS
✔ Mandatory in real apps

---

## 8️⃣ Form Validation Example (Complete)

```php
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {

    if (empty($_POST["name"])) {
        echo "Name is required<br>";
    } else {
        $name = htmlspecialchars($_POST["name"]);
        echo "Name: $name<br>";
    }

    if (empty($_POST["email"])) {
        echo "Email is required<br>";
    } else {
        $email = filter_input(INPUT_POST, "email", FILTER_VALIDATE_EMAIL);
        if ($email) {
            echo "Email: $email<br>";
        } else {
            echo "Invalid email format<br>";
        }
    }
}
?>
```

## 9️⃣ Sticky Form (Preserve Values)

```html
<input type="text" name="name" value="<?php echo $_POST['name'] ?? ''; ?>">
```

✔ User doesn’t lose input on error
✔ Professional UX
