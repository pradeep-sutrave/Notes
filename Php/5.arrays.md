

# 📘 UNIT – 5 : ARRAYS IN PHP (DETAILED)

Arrays allow us to **store multiple values in a single variable**.
Without arrays, **real PHP applications are impossible**.

---

## 1️⃣ What is an Array?

### Definition (Exam-ready ✍️)

> An array is a data structure that stores multiple values in a single variable.

### Why Arrays?

Without array:

```php
$a = 10;
$b = 20;
$c = 30;
```

With array:

```php
$numbers = [10, 20, 30];
```

✔ Cleaner
✔ Easier to manage
✔ Loop-friendly

---

## 2️⃣ Types of Arrays in PHP

PHP supports **three main types of arrays**:

1️⃣ Indexed Array
2️⃣ Associative Array
3️⃣ Multidimensional Array

# 🔹 1. Indexed Arrays

👉 Index is **numeric** (0, 1, 2, …)

### Declaration

```php
$subjects = ["DSA", "DBMS", "OS", "CN"];
```

### Accessing Elements

```php
echo $subjects[0]; // DSA
echo $subjects[2]; // OS
```

---

### Looping Indexed Array

```php
for ($i = 0; $i < count($subjects); $i++) {
    echo $subjects[$i] . "<br>";
}
```

---

### Common Indexed Array Functions

| Function          | Purpose            |
| ----------------- | ------------------ |
| `count()`         | Number of elements |
| `array_push()`    | Add at end         |
| `array_pop()`     | Remove last        |
| `array_unshift()` | Add at beginning   |
| `array_shift()`   | Remove first       |
| `sort()`          | Ascending sort     |
| `rsort()`         | Descending sort    |
| `in_array()`      | Search value       |

---

# 🔹 2. Associative Arrays

👉 Uses **key ⇒ value** pairs.

### Declaration

```php
$student = [
    "name" => "Bhupathi",
    "branch" => "CSE",
    "year" => 2
];
```

### Accessing Values

```php
echo $student["name"];
echo $student["branch"];
```

---

### Looping Associative Array

```php
foreach ($student as $key => $value) {
    echo $key . " : " . $value . "<br>";
}
```

---

### Associative Array Functions

| Function             | Purpose              |
| -------------------- | -------------------- |
| `array_keys()`       | Get all keys         |
| `array_values()`     | Get all values       |
| `array_key_exists()` | Check key            |
| `asort()`            | Sort by value (ASC)  |
| `arsort()`           | Sort by value (DESC) |
| `ksort()`            | Sort by key (ASC)    |
| `krsort()`           | Sort by key (DESC)   |

---

# 🔹 3. Multidimensional Arrays

👉 An array **inside another array**
Used for tables, records, database results.

### Declaration

```php
$students = [
    ["Bhupathi", "CSE", 8.5],
    ["Rahul", "ECE", 7.9],
    ["Anita", "IT", 8.8]
];
```

### Accessing Elements

```php
echo $students[1][0]; // Rahul
```

---

### Looping Multidimensional Array

```php
foreach ($students as $student) {
    foreach ($student as $data) {
        echo $data . " ";
    }
    echo "<br>";
}
```

---

### Multidimensional Array Functions

(No special functions — use loops + utility functions)

| Function         | Use            |
| ---------------- | -------------- |
| `count()`        | Count rows     |
| `array_column()` | Extract column |
| `array_merge()`  | Merge arrays   |

---

# 🔹 4. Common Array Utility Functions (🔥 VERY IMPORTANT)

These work with **all array types**.

| Function            | Purpose           |
| ------------------- | ----------------- |
| `array_merge()`     | Merge arrays      |
| `array_unique()`    | Remove duplicates |
| `array_reverse()`   | Reverse array     |
| `array_slice()`     | Extract part      |
| `array_splice()`    | Remove/replace    |
| `array_diff()`      | Difference        |
| `array_intersect()` | Common values     |
| `explode()`         | String → Array    |
| `implode()`         | Array → String    |
