This unit teaches PHP to **store, fetch, update, and delete data** from a database.

## 1️⃣ What is MySQL?

### Definition (Exam-ready ✍️)

> MySQL is an open-source **relational database management system (RDBMS)** used to store and manage data.

## 2️⃣ Why PHP Needs MySQL?

PHP alone:

* Temporary data
* Lost on refresh ❌

PHP + MySQL:

* Permanent storage ✅
* Login systems
* Dashboards
* Records management

## 4️⃣ Creating a Database & Table

### Create Database

```sql
CREATE DATABASE college;
```

### Use Database

```sql
USE college;
```

### Create Table

```sql
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50),
    email VARCHAR(50),
    branch VARCHAR(20)
);
```

---

## 5️⃣ Connecting PHP with MySQL

PHP supports **two ways**:
1️⃣ `mysqli` (Procedural / OOP)
2️⃣ `PDO` (Advanced)

👉 We start with **mysqli (procedural)**.

---

### MySQL Connection File (`db.php`)

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "college");

if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
?>
```

✔ `localhost` → server
✔ `root` → username
✔ `college` → database

---

## 6️⃣ INSERT Data (Create)

```php
<?php
include "db.php";

$sql = "INSERT INTO students (name, email, branch)
        VALUES ('Bhupathi', 'b@gmail.com', 'CSE')";

if (mysqli_query($conn, $sql)) {
    echo "Record inserted successfully";
} else {
    echo "Error: " . mysqli_error($conn);
}
?>
```

---

## 7️⃣ SELECT Data (Read)

```php
<?php
include "db.php";

$result = mysqli_query($conn, "SELECT * FROM students");

while ($row = mysqli_fetch_assoc($result)) {
    echo $row["id"] . " ";
    echo $row["name"] . " ";
    echo $row["email"] . " ";
    echo $row["branch"] . "<br>";
}
?>
```

---

## 8️⃣ UPDATE Data

```php
<?php
include "db.php";

$sql = "UPDATE students SET branch='IT' WHERE id=1";

mysqli_query($conn, $sql);
?>
```

---

## 9️⃣ DELETE Data

```php
<?php
include "db.php";

$sql = "DELETE FROM students WHERE id=2";

mysqli_query($conn, $sql);
?>
```

## 1️⃣2️⃣ SQL Injection (🔥 VERY IMPORTANT)

### Dangerous Input

```
' OR '1'='1
```

This can:

* Bypass login
* Delete data

❌ Never trust user input

---

## 1️⃣3️⃣ Prepared Statements (SECURE WAY)

```php
<?php
$stmt = mysqli_prepare(
    $conn,
    "INSERT INTO students (name, email, branch) VALUES (?, ?, ?)"
);

mysqli_stmt_bind_param($stmt, "sss", $name, $email, $branch);
mysqli_stmt_execute($stmt);
?>
```

✔ Prevents SQL Injection
✔ Best practice

## 1️⃣4️⃣ Password Hashing (Login Systems 🔐)

```php
$hash = password_hash("1234", PASSWORD_DEFAULT);
```

Verify password:

```php
password_verify("1234", $hash);
```

❌ Never store plain passwords
