
# 📘 UNIT – 3 : OPERATORS IN PHP (DETAILED)

Operators are **symbols that perform operations on variables and values**.
This unit is **very important for logic building + exams**.

---

## 1️⃣ What is an Operator?

👉 An **operator** tells PHP **what action to perform**.

Example:

```php
$a = 10 + 5;
```

* `+` → operator
* `10`, `5` → operands

---

## 2️⃣ Types of Operators in PHP

PHP operators are classified into:

1. Arithmetic Operators
2. Assignment Operators
3. Comparison Operators
4. Logical Operators
5. Increment / Decrement Operators
6. String Operators
7. Ternary Operator
8. Operator Precedence

---

## 3️⃣ Arithmetic Operators

Used for **mathematical calculations**.

```php
<?php
$a = 10;
$b = 3;

echo $a + $b; // 13
echo "<br>";
echo $a - $b; // 7
echo "<br>";
echo $a * $b; // 30
echo "<br>";
echo $a / $b; // 3.33
echo "<br>";
echo $a % $b; // 1
?>
```

| Operator | Meaning             |
| -------- | ------------------- |
| `+`      | Addition            |
| `-`      | Subtraction         |
| `*`      | Multiplication      |
| `/`      | Division            |
| `%`      | Modulus (remainder) |

---

## 4️⃣ Assignment Operators

Used to **assign or update values**.

```php
<?php
$x = 10;
$x += 5; // 15
$x -= 2; // 13
$x *= 2; // 26
$x /= 2; // 13
?>
```

| Operator | Meaning           |
| -------- | ----------------- |
| `=`      | Assign            |
| `+=`     | Add & assign      |
| `-=`     | Subtract & assign |
| `*=`     | Multiply & assign |
| `/=`     | Divide & assign   |

---

## 5️⃣ Comparison Operators (🔥 VERY IMPORTANT)

Used to **compare two values**.
Result is **true or false**.

```php
<?php
$a = 10;
$b = "10";

var_dump($a == $b);  // true
var_dump($a === $b); // false
?>
```

| Operator | Meaning                  |
| -------- | ------------------------ |
| `==`     | Equal (value only)       |
| `===`    | Identical (value + type) |
| `!=`     | Not equal                |
| `!==`    | Not identical            |
| `>`      | Greater than             |
| `<`      | Less than                |
| `>=`     | Greater or equal         |
| `<=`     | Less or equal            |

📌 **Interview favorite:**
Difference between `==` and `===`

---

## 6️⃣ Logical Operators

Used to **combine multiple conditions**.

```php
<?php
$age = 20;
$hasId = true;

if ($age >= 18 && $hasId) {
    echo "Entry allowed";
}
?>
```

| Operator | Meaning |   |    |
| -------- | ------- | - | -- |
| `&&`     | AND     |   |    |
| `        |         | ` | OR |
| `!`      | NOT     |   |    |

---

## 7️⃣ Increment & Decrement Operators

Used to **increase or decrease values by 1**.

```php
<?php
$x = 5;

echo ++$x; // 6 (pre-increment)
echo "<br>";
echo $x++; // 6 (post-increment)
echo "<br>";
echo $x;   // 7
?>
```

| Operator | Meaning         |
| -------- | --------------- |
| `++$x`   | Increment first |
| `$x++`   | Increment later |
| `--$x`   | Decrement       |

---

## 8️⃣ String Operators

Used to **combine strings**.

```php
<?php
$a = "Hello";
$b = "PHP";

echo $a . " " . $b;
?>
```

📌 PHP uses **`.` (dot)** for concatenation
❌ `+` does NOT work for strings

---

## 9️⃣ Ternary Operator

Shortcut for `if–else`.

```php
<?php
$age = 16;
echo ($age >= 18) ? "Adult" : "Minor";
?>
```

Syntax:

```
(condition) ? value_if_true : value_if_false;
```

---

## 🔟 Operator Precedence

Defines **which operator executes first**.

Example:

```php
<?php
echo 10 + 5 * 2;
?>
```

Output:

```
20
```

Why?
👉 `*` has higher precedence than `+`

---

## ✍️ UNIT–3 EXAM SHORT NOTES

* Operators perform actions on operands
* `===` checks value + type
* Logical operators combine conditions
* `.` is used for string concatenation
* Ternary operator is shorthand for if–else
